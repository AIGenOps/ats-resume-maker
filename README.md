# ATS Resume Maker

A powerful, free, and client-side-only web tool to build, customize, and export high-fidelity, ATS-friendly resumes in professional layouts.

No signup, no databases, no subscriptions. Your data stays 100% private in your browser.

---

## Key Features

*   **Four Visual Templates**:
    *   **Standard Corporate (`classic`)**: Centered header, single column, solid lines—perfect for traditional industries.
    *   **Modern Creative (`twocolumn`)**: Two-column layout with a circular avatar photo, skill pills, and language proficiency dots.
    *   **Minimal Clean (`minimalist`)**: Clean layout with stacked contact details, dashed section dividers, and grid-aligned skills.
    *   **Executive Timeline (`timeline`)**: Profile avatar on the left, right-aligned narrative section with a vertical timeline node trace, and progress bar metrics for languages.
*   **Smart "Page-Fit" Auto-Scaler**: Toggleable scaling logic that automatically adjusts margins, line heights, and font sizes to ensure your resume fits perfectly on a single page or fits cleanly without awkward page-breaks.
*   **Quality & Formatting Auditor**: Real-time analysis of your writing that flags passive language, checks date formatting consistency, highlights excessive bullet lengths, and verifies action verb usage.
*   **ATS Keyword Scanner**: Scan your resume against a target job description to match keywords and optimize your ATS score.
*   **High-Fidelity jsPDF Exporter**: Custom PDF generation engine that supports two-column wraps, image rendering, custom rating meters, and precise grid drawing.
*   **Local Project Manager**: Save, export, and load multiple resume configurations directly using browser `localStorage` or backup JSON files.
*   **Sleek Modern UI**: Premium glassmorphic interface with seamless Light and Dark mode toggles.

---

## How to Use

1. Visit the live tool: **[ATS Resume Maker](https://aigenops.github.io/ats-resume-maker/)**
2. Edit fields in the sidebar panels, choose your template style, and customize fonts/margins.
3. Click the dominant **Export jsPDF** button in the bottom toolbar to generate and download your ATS-ready resume!
